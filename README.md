\# Multi-Agent Prompt Factory



一个基于 CrewAI 的多 Agent Prompt 生成器。



\## 功能



\- 分析用户需求

\- 生成 Prompt 初稿

\- 审查 Prompt 问题

\- 输出最终优化版 Prompt

\- 支持 Docker 运行



\## 技术栈



\- Python

\- CrewAI

\- LiteLLM

\- Docker

\- python-dotenv
